Time Sources (GML Visual)

This page lists the GML Visual actions available for Time Sources. Read an overview of Time Sources here.

The following actions are available for Time Sources in GML Visual:

  • Create Time Source
  • Destroy Time Source
  • Start Time Source
  • Stop Time Source
  • Pause Time Source
  • Resume Time Source
  • Reconfigure Time Source
  • Reset Time Source
  • Get Children
  • Get Parent
  • Get Period
  • Get Reps Completed
  • Get Reps Remaining
  • Get State
  • Get Time Remaining
  • Get Units
  • If Time Source Exists
  • If Time Source Parent
  • If Time Source State
  • If Time Source Units

 

The following actions are provided for converting time periods:

  • Seconds to BPM
  • BPM to Seconds

 

The following constants are used with Time Source actions:

  • Built-In Time Sources
  • Time Source Units
  • Time Source Expiry Types
  • Time Source States
