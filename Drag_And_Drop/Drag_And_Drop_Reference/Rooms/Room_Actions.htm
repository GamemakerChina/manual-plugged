Room Action Library

The Room library permits you to change and check game rooms, based on the order defined in the Room Manager. Note that all actions which change the room will also trigger the Other - Room Events .

The following actions are available:

 

Go To Room
Go To Next Room
Go To Previous Room
Restart Room
If Room Is First
If Room Is Last
Get Current Room
