window_get_borderless_fullscreen

This function returns whether a borderless window for fullscreen mode is enabled or not.

NOTE This function only works on Windows.

 

Syntax:

window_get_borderless_fullscreen();

 

Returns:

Boolean

 

Example:

var _borderless = window_get_borderless_fullscreen();
show_debug_message($"Borderless Fullscreen Enabled: {_borderless}");

The code above calls window_get_borderless_fullscreen to find out if borderless fullscreen is currently enabled. It then shows a debug message.
