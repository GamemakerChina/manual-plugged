audio_group_load

This function will load all the sounds that are flagged as belonging to the given Audio Group into memory.

The function will return true if loading is initiated and false if the group ID is invalid, or it has already been flagged for loading.

NOTE The function is asynchronous so your game will continue to run while the audio is loaded in the background. This means that it will also trigger an Asynchronous Load/Save Event to inform you that the whole group has been loaded into memory and the sounds can now be used.

 

Syntax:

audio_group_load(groupID);

ArgumentTypeDescription
groupIDAudio Group IDThe index of the audio group to load (as defined in the Audio Groups window)

 

Returns:

Boolean

 

Example:

if (!audio_group_is_loaded(audiogroup_level1))
{
    audio_group_load(audiogroup_level1);
}

The above code checks to see if an audio group has been loaded and if not, it loads it.
